
# Challenge One Conversor de Moneda

Proyecto Conversor de monedas desarrollado en java para el Challenge de Alura One

## Características

- Conver-sor de dolar a pesos
    (Chile CLP,
    Argentino ARS,
    Brasil BRL,
    Colombia COP,
    Peru SOL)
-   Conversión inverso
-   Historial
